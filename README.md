# imworking

Script that reads a file where several logs are stored and output them to appear to be reviewing them

Create a matrix programmer! Everything is possible with this tool, just fill in the file with the data to be displayed randomly: D

I am not responsible if you are fired in your company ...

# How to use it

1. clone the repository
2. Fill in the log.db with any type of content that you want to be displayed randomly (more lines, more real it seems)
3. run init.py
4. Enjoy your free time, you're welcome;)
